# Linux/Windows/Mac ErgoDox Layout
The goal of this mapping is to have to reduce the number of different shortcuts one would have to remember for different
OS's. Some decisions were made to with Jetbrains IDE's shortcuts in mind as well.

## Building
The following steps will allow you to compile a hex file for this layout and flash your ErgoDox.

Setup your build Env
1. `python3 -m venv venv`
1. `source ./venv/bin/activate`
1. `pip install qmk`

Setup default keyboard and keymap
1. `qmk config user.keyboard=ergodox_ez`
1. `qmk config user.keymap=maverick1872`

Build a hex file to be used when flashing
* `qmk compile`

Alternatively you can use the following if you want to build a non default keymap
* `qmk compile -kb 'keyboard_folder_name' -km 'keyboard_mapping_folder_name'`


#Layer Mappings
`MO` denotes a Momentary layer toggle  
`TD` denotes a Tap Dance key  
`TO` denotes a Persistent layer toggle

#### TD Definitions
`TD ;` - `; on single tap, : on double tap`  
`TD '` - `' on single tap, " on double tap`  
`TD /` - `/ on single tap, ? on double tap`  
`TD ENT` - `Enter on single tap, . on double tap`
